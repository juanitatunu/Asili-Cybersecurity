Cybersecurity Threats

Viruses: Malicious software that attaches to a host file and can spread when the file is executed.

Worms: Standalone malicious programs that replicate themselves to spread to other computers.

Trojan Horses: Malicious software disguised as legitimate software.

Payloads: The part of malware that performs the malicious action.

Access to a system or data without permission.

Stealing personal or sensitive information from individuals or organizations.

Attempting to acquire sensitive information by pretending to be a trustworthy entity in electronic communication.

Malware that encrypts a victim's files and demands payment for the decryption key.
